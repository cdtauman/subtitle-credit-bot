# 🎬 בוט טלגרם להוספת קרדיט לכתוביות

בוט טלגרם מתקדם להוספת שורות קרדיט אוטומטיות לקבצי כתוביות SRT.

---

## ✨ תכונות עיקריות

- 📂 תמיכה בקבצי `.srt` בודדים וארכיוני `.zip`
- 🎨 הגדרת צבע, גופן ומיקום לקרדיט
- ⏱️ קרדיטים בתחילת, אמצע וסוף הסרט עם תזמון מותאם
- 🧠 מניעת התנגשויות חכמה תלוית-מיקום
- 👑 מערכת ניהול מלאה (אישור, חסימה, קידום למנהל, שידור הודעות)
- 💾 שמירת הגדרות משתמש במסד נתונים SQLite
- 🔤 קידוד UTF-8 מלא
- 📋 תפריט מקלדת קבועה

---

## 🚀 התקנה והפעלה

### 1. שכפול הפרויקט
```bash
git clone <repo-url>
cd telegram-subtitle-bot
```

### 2. התקנת תלויות
```bash
pip install -r requirements.txt
```

### 3. הגדרת משתני סביבה
```bash
cp .env.example .env
```
ערוך את `.env` ומלא את הפרטים:
- `BOT_TOKEN` - טוקן מ-BotFather
- `ADMIN_IDS` - מזהה הטלגרם שלך (קבל דרך @userinfobot)

### 4. הפעלת הבוט
```bash
python main.py
```

## 👤 תזרים משתמש

1. **משתמש חדש** - שולח `/start`, הבקשה עוברת לאישור מנהל
2. **אחרי אישור** - תהליך הגדרה ראשונית 8 שלבים
3. **שימוש שוטף** - שליחת קבצי `.srt` / `.zip` לעיבוד אוטומטי
4. **הגדרות** - שינוי בכל עת דרך ⚙️ הגדרות
5. **חד-פעמי** - 🎬 כתוביות מותאמות אישית לעיבוד בודד שונה

## 👑 פקודות מנהל

| פקודה | תיאור |
|-------|--------|
| `/ban [user_id]` | חסימת משתמש |
| `/promote [user_id]` | קידום למנהל |
| `/broadcast [הודעה]` | שידור לכל המשתמשים |
| `/users` | רשימת כל המשתמשים |

## ⚙️ לוגיקת תזמון קרדיטים

### קרדיט פתיחה
מופיע בשניות 5 ממשך הסרט, לפי משך שהוגדר.

### קרדיט אמצע
חוזר כל X דקות (לפי הגדרת תדירות), לפי משך שהוגדר.

### קרדיט סיום
מופיע לפני שתי הכתוביות האחרונות של הקובץ (להימנעות מהסתרת קרדיטים מקוריים).

### מניעת התנגשויות (חכמה תלוית-מיקום)
- **קרדיט למעלה + כתוביות למטה** → אין בדיקת התנגשות, חפיפה מותרת
- **קרדיט למטה + כתוביות למטה** → הבוט מזיז את חלון הזמן קדימה עד למציאת חלון פנוי
